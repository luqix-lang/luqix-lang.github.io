Download Luqix programming Language

Luqix is in early development with regular nightly builds so if you need the very current verion you may have to compile it yourself.

Compiling source code

The luqix source code is hosted on github .

1. Download and install any dlang compiler.

2. Get source code from github and unzip it.

3. Enter luqix folder and run make command

~$ make

Watch tutorial for compiling luqix.

Download binaries

Mac and some other platforms binary builds are not yet provided but one can compile a build on any of those platforms then share with the community and us the developing team. I expect no errors during compilation.
