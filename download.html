Download Luqix programming Language

Note
Luqix is a one person project, built on personal and sorted suggested preferences. It is so buggy now, but the aim is a language that able to accomplish alot. The bugs cease as time passes and lots will be brought to the table.

Compiling source code

The luqix source code is hosted on github .

1. Download and install any dlang compiler.

2. Get source code from github and unzip it.

3. Enter luqix folder and run make command.

~$ make

4. An executable file luqix or luqix.exe will be formed in pwd.

5. Thats the interpreter so test with cmd luqix.exe windows or ./luqix unix.

~$ luqix.exe

Note
An additional step for linux users may be required to change the executable permissions, do as follows and please windows users report bugs as the program is developed more on windows, wanna know why? contact me!

~$ sudo chmod +x luqix

Watch tutorial for compiling luqix.
