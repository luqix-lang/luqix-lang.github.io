luqix-lang

Source code installation

Since the project is in early development, it has no official way to distribute it. So we recommend compiling the source code. To get a much stable executable for your system.

install dmd compiler to compile Luqix source code at dlang.org

download Luqix source code from our github repo

unzip the archive and change to the new luqix directory

run the make command to produce an executable
