Wanna build some simple GUI.

Its of no doubt that building a website with html and css is very easy and when you add javascript, you have no limit. So using javascript web-socket can give us some simple GUI capabilities.

So we are providing a module called ws for now that makes a simple websocket to share data with javascript and the browser window. It is hard coded in Dlang and all it does is to pass and receive data with masking it.

Now i expect a good programmer to build some modern GUI with this feature using Luqix has the backend doing things limited to the browser cheaply.

Lets try it out.

// Luqix side let ws = importc('websocket') str = importc('string') bytes = importc('bytes') conn = ws.WebsocketServer('localhost', 7071) print('Websocket server up at address ws:/localhost:7071/') fn greet(name){ n = bytes.decode(name) print(f'How are you {n}') conn.send(str.encode(f'thanks for connecting {n}.')) } fn hey(){ print('Successful handshake....') conn.send(str.encode('Hello from Luqix')) } conn.onopen(hey) conn.onmessage(greet) conn.start()
