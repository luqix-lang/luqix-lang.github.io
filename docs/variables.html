Basics-variables

There are a few ways to make variables.

A variables is a value that can change, depending on conditions or on information passed to the program.

No special where to create variables just the normal.

food = 'banana' amount = 500 shopping_lists = ['oil', 'vinegar', 'onions', 'charcoal']

Using the setvar( ) function in the sys module.

This method is just doing the same as the above just that for it its dynamic.

let sys = importc('sys'); sys.setvar('animal', 'horse'); print(animal) // horse

Just remember variables are limited to the scope they are created in here. There is no provision for GLOBAL variables like other laguages do.
