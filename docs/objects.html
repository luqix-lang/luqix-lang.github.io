# making types

This is simply defining a class, the class name will become the object data type and any functions or variables attached to a self identifier will become the class attributes.

The function __init__ may or not be defined for an object but it is executed when every a new object instance is to be made.

class Student { level = 'III' fn __init__(name, age, scores) { self.name = name self.age = age self.scores = scores } fn hey(){ self.age = self.age + 0 // just a use case return f'hey {self.name}' } }

We have created a type called Student and by using the type(..) function we see its of type student similar to the list, strings are others.

print(type(Student)) // Student

So thats a step to creating a new type, now lets have some of its instances and manuplate their data.

one = Student('Brian', 20, 18) two = Student('Alice', 19, 20) grade = [one, two] // [Student (object), Student (object)] for (i; i < len(grade); i++) print(grade[i].name)

Getting one of the attributes and using them anywhere its a piece of cake.

rnd_student = f'My name is {two.name}, am {two.age} and I scored {two.scores} points.' print(rnd_student) // My name is Alice, am 19 and I scored 20 points.
