random module

Module random

This module deals with generation of random data and choosing random items out of lists.

# shuffle(list n)

Shuffles items in list n and so the list changes in all its occurences.

n = [1,2,3,4,5,6] random.shuffle(n) print(n) // [2,4,1,3,6,5] or something else

# sample(list n, int num)

Returns a sample of items in list n shuffled of length num.

n = ['a','b','c','d','e'] samp = random.sample(n, 3) print(samp) // could be ['c','a','d']

# random()

Returns a random floating number or atleast 10 decimal places.

print(random.random()) // 0.00357

# uniform(float min, float max) *

Generates a Continuous uniform distribution and returns a floating number.

print(random.uniform(0, 70))

# interger(int min, opt int max)

Return the continous uniform distribution but returns a whole number.

print(random.interger(4, 11)) // either 4,5,6,7,8,9,10

# bool()

Luckily return true or flase.

# pick(list n)

Random picks and returns one item from list n.

n = [1, 2, 'a', 'b', 'b', true, false] print(random.pick(n)) // could be 2, 'a' or ... in n

# string(int length, opt string specials)

Generates a random string from all alphabetical letters of length length and could add special characters or imojis optionally passed in the specials argument. Can reuse a letter.

pwd = random.string(8, '45678%') // could be er5KiKy4

# string_pool(string pool, int length) *

Takes in a string pool that it uses to generate random strings of length length as specified.

color = random.string_pool('abcdef0123456789'); print(f'#{color}') // #aef077
