Basics-while

The while loop is here too, just similar to that of the olden language also.

A "While" Loop is used to repeat a specific block of code an unknown number of times, until a condition is met.

Just like other while loops, a condition is set and if its met, then it will stop unless you break the loop. First we must write the while keyword followed by the condition and the demacate the loop's scope.

while (true) { // never stops print('I will run forever') }

Since the condition is always true, then the loop never stops. So next we shall base on a condition that will be interrupted in sometime.

count = 1000 while (count) { print(count) // subtracts one from count count = count-1 }

No it stops when count is 0 since 0 is false in some programming ascepts.

break the loop

We shall break the loop using the break, well you will base on a condition.

And also still the loop creates no new scope, so its variables are put in its parent scope.
