The luqix file?

This is a file ending with .lqs file extension and it is better it is reserved in there own folder to demacate your application and facilitate modulization.

app.lqs

Did you know? With this .lqs extension is how the language is able to locate modules. So other files it is a miracle for them to work.

Executing the file.

Here we now call the interpreter executable first followed by the filename and endavour to be in the right path.

luqix app.lqs ./luqix app.lqs
