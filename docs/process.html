Process module

Module process

This module is used to run system process and spawn shells but for real its not stable yet. Pre-usage is required and please report bugs on any of our reach, thank you.

Module attributes.

run

browse

Popen

$ run(string cmd) *

Executes a command and returns its output after the process is done, ut this is very unstable..

process.run('echo hello')

$ browse(string cmd) *

Open given url string in a new default bwoser window.

process.browse('luqix-lang.github.io')

$ Popen(string cmd, enum{shell=true, stdout=filename, stdin, stderr}) *

Opens a pipe object and a wide range of controlling background process.

This too is very unstable.

pipe = process.Popen('ls -l', enum {shell=true}) print(pipe.fetch())

pipe.wait()

pipe.kill()
