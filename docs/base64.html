Module Base64

Base64 is a group of binary-to-text encoding schemes that represent binary data in sequences of 24 bits that can be represented by four 6-bit Base64 digits.

This module encodes and decodes to base64 data with a very simple API but be aware not to give wrong data as it will result into a segmentation fault (core dumped) atleast for now.

The base64 API.

encode64

decode64

encode32

decode32

Warning The 32 encoding is custom made and built just no the principles, so it may not be the safest to use. atleast for now.

$ encode64(bytes data) *

Encodes bytes to a base64 string, returning the product.

print(base64.encode64(b'God is good'))

$ decode64(base64 string data) *

Decodes base64 string data to bytes data.

print(base64.decode64('aGVsbG8gd29ybGQh'))

$ encode32(bytes data) *

Encodes bytes to a base32 string, returning the product.

print(base64.encode32(b'God is good.'))

$ decode32(base32 string data) *

Decodes base32 string data to bytes data.

print(base64.decode32('NBSWY3DPEB3W64TMMQQQ===='))
