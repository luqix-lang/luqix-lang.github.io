locals module

locals module has very essential functions and so they are even already all imported to the file scope. They do tasks dealing with data being used in the runtime and also controlling part of the runtime

let locals = require('locals');

Summarised table

Function name Usage and Description
Exception Exception error thrown requiring a luqix enum with items message, type, line, Exception(enum {message='Wrong usage', type="NormalError", line=0})
attr Returns a list of all attributes of the given data, attr(str).
bytes Its the bytes object for handling bytes data and creating, bytes().
read Reads the whole file returning all its string data read('file.txt') without opening it.
delattr Deletes the specified attribute of the given data, delattr(str, 'join').
dict Its the object that handles dictionaries and creating new ones, dict(). Plus casting to dictionaries.
eval Evaluates the given string code, in the calling scope, eval('1 + 3').
exec Executes string code by injecting it in the runtime, exec('a = "hello world"; print(a);')
exit Stops calling thread execution, exit().
getattr Dynamically gets and returns attribute of any data, getattr(str, 'join').
len Returns the length of given data, len([1,2,3,4,5,6]) as an interger (size_t).
list Its the object for handling and casting to lists, list('hello') plus making them.
numb Its the number object for handling numbers numb('0.3') and casting to numbers.
print Prints strings to the standard output, print('hello world!').
prompt Gets user input from console, prompt("Write: "), and takes an optional string for prompting.
require Imports native modules to the scope, reusingly by their string name, require('math').
setattr Sets new attributes to the given data if possible, setattra(str, 'added', 'am new').
str Its the object for handling and creating strings, str(12), plus casting to strings.
super Object for handling, class inheritance but not ready yet.
type Returns the type of the given data as a string, type([1,2,4]).

Note
All functions for handling a specific datatype are also incharge of casting to that datatype any data given to them.
