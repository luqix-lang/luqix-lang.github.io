locals module

Module locals

This builtin module has special functions for the interpreter, just navigate.

The builtins module is automatically loaded every time Luqix interpreter starts, either as a top level execution environment or as interactive session. The commonly used functions used without importing from any where are defined in this module.

Even when you import a module, this module is also imported to the modules's scope.

# print(args, ...)

prints the given arguments string formats on the console or terminal.

Takes in any nnumber of arguments.

print('hello world', 19)

# attr(object)

Returns the a list of attributes in string format of the object specified.

Takes only one argument and it should be given.

let rd = importc('random') n = attr(rd) // ['choice', 'num', ...]

# importc('name')

Imports a builtin module from the interpreter's core if its at ground state or returns the already called module if its already in use.

Takes only one argument and it should be given, this is the module name.

let rd = importc('random') math = importc('math')

Its limited to builtin modules only and for normal Luqix files, use the import statement.

# prompt('optional')

Reads user input from the terminal or console.

Takes only one argument and its optional, this should be a string that prompts the input.

pwd = prompt('Enter password: ')

The argument given is simply a placeholder but not some default text and there is no provision from default input, an if statement and len method could help with that.

# len(obj)

Returns the length of the object given, and if no length is specified, then a 0 is returned.

Takes only one argument and it should be given.

n = len('Jesus is Lord')

# type(obj)

Returns the type of data given, it could be a custom object or builtin datatypes.

Takes only one argument and it should be given.

type('hello world') // string type(1284) // number

The type is returned as a string so comparision must be with a string.

# exit()

Terminates/stops the execution of the calling scope or thread.

exit()

No arguments are need. And this uses the C function std.exit taking in argument 0.

# delattr(obj, 'attr-name')

Deletes the specified attribute from the given object.

delattr(math, 'cos')

# setattr(obj, 'attr-name', value)

Sets a new attribute to the specified object.

setattr(math, 'pi', (22/7))

# getattr(obj, 'attr-name')

Returns the specified attribute values of the given object.

cosine = getattr(math, 'cos')

# super(obj, arg)

appends attribute to the object given as parameter one.

// obj imagine its a class that has inherited parent class of arg super(obj, arg)
