Locals module

locals is a builtin module that is readily imported to every new scope formed by the interpreter since it holds greater functionality that must be at hand. This is similar to python's builtins module having functions like printing to the console, getting data length and much more.

Table of functions

attribute function
print writes to standard output or screen
prompt gets user input from standard input
len gets the length of a datatype
attr lists all attributes of the given data
exit terminates execution of calling process
require imports and then return a builtin module
getattr gets data attribute dynamically
setattr sets or replace data attribute dynamically
delattr deletes an attribute from data.
Exception returns a recognisable error object for luqix
super handles inheritance unpackaging.
