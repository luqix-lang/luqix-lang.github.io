Fs module

fs module deals with the Os file system handling the likes of reading and writing data to files and files locations on the Pc. This is a convient module has it includes low level file writing with usage of bytes and it functions come from direct C functions.

let fs = require('fs');

# file.openFile(str filename, string mode)

Opens a file in the specified mode and returns an object stream to which you can manipulate the file data. Talking of modes, they are quite a number. These specify how the file should be handled, either in string mode or bytes mode.
