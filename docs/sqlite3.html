Module Sqlite3

The websocket module is a builtin module that allows one to create the C based sqlite3 database and can easily connect to exisiting ones and also update the data.

This module is a wrapper for the real C sqlite3 source code and library and so for its usage or compilation, there is need to have the sqlite3 library on your system. For linux is easy but windows its hard.

Info

For now, we can use this module to build browser apps backend by Luqix.
