Luqix Keywords

Luqix keywords are special reserved words that specify different features in code being written. These keywords are not usable in your code from where they are suppossed.

Improper usage of these keywords in your code is spotted at parsing time even before code execution starts and a SyntaxError is raised.

Luqix runtime has only 30 keywords, each doing different things.
