Basics-functions

Functions are one of the most most useful, features in coding.

Functions are "self contained" modules of code that accomplish a specific task.

No special where to create functions that differs from the other languages. It has a name, takes parameters and demacates its own code and also returns a value.

fn fn_name(param1, param2) { print(param1) return param2; }

return keyword.

The return key word specifies a value the function returns from its scope, if not given, it defaults to none value.

Return also acts as break in a loop or a function meaning that if its satisfied, it stops executing the calling scope and return the values. So don't call it from the global scope, the program will terminate.

fn count(n) { for(let i=0; i < n; i++) { print(i) if (i == 19) return i } } print(count(7)) // none print(count(21)) // 19

returns none if there is no reaching 19 or returns 19
