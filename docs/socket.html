Socket module

socket provides networking to a program using the Berkeley Socket Interface that is commonly seen in many programming language. Docs like for python socket module will greatly help as no change was possible for this module in luqix implimentation.

Note!
Variables in this module are simply numbers, globally know for Berkeley sockets, so their existance in the module could seize, to keep the module clean. But always, a mapping will be found here.

socket.Socket(AF, SOCK)

Creates a socket object ready for interfacing with connection of the type specified by the address family and socket type which would include TCP, UDP and many more.

conn = socket.Socket(socket.AF_INET, socket.SOCK_STREAM) // tcp socket
attribute function
accept accepts connection to the server returning an individual client handling object.
addr_family returns the string address family of the stream
bind binds the socket object to the given localhost and port
close closes the connection
connect connects socket object to a server at the given localhost and port.
hostname returns the hostname string used by the server.
isalive returns true if the connection is still running or false.
listen determines the maximum number of clients the server should listen to at a time.
send sends bytes data on the network
sendto sends byte data on the network like UDP that requires no validation.
setblocking if given true set socket to blocking or false to not blocking.
setsockopt Helps in reusing the port and more.
shutdown shutdowns connections to the socket either for one side or all.
