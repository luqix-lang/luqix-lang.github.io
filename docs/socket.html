Socket module

socket provides networking to a program using the Berkeley Socket Interface that is commonly seen in many programming language. Docs like for python socket module will greatly help as no change was possible for this module in luqix implimentation.

host = s.accept()

Accept incoming connection and wait if blocking, returns new connected host streaming object.

a = s.addressFamily

Returns address family used by the socket.

a = s.alive()

Returns true if socket is still up.

s.bind('localhost', 7575)

Binds socket to given hostname and server.

b = s.blocked()

Returns true if socket is blocking.

s.close()

Closes the socket connection.

s.connect('localhost', 7575)

Connects socket to given hostname and port.

a = s.hostname

Returns socket hostname.

s.listen(5)

Sets socket to listen to given number of hosts.

data = s.recieve(1024)

Returns sent bytes data from other connection sender.

s.send(b'hello world')

Send sbytes data over network.

s.sendTo(b'hello world')

Sends bytes data over UDP network.

s.setBlocking(true)

Sets socket to or from blocking.

s.setSocketOption(2, 1)

Helps in port reusing.

s.shutDown(1)

Shuts down or stops abilities of a socket like listening, sending or both.
