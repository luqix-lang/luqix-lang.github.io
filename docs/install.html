Installation

Hey, installation is Luqix is not the most favourable since the language's development is very early, so there is not yet a favourable way to publish.

And due to lack of computers of different systems to test on, we are to compile the source code. But any support or contribution for a system its welcome, contact me

The project is hosted on GITHUB

Requirements

Since luqix is builtin in dlang, we shall need its compiler.

Download DMD compiler.

Download LDC compiler, this one is not guaranted to work.

Downlod and install

Download or clone from github follow link

git clone https://github.com/luqix-lang/luqix.git

Unpack the archive and then open terminal.

Change to the new luqix directory.

cd luqix

At this point, there is no dependencies needed expect the DMD dlang compiler and its requirements.

Now run the make command to get the luqix executable.

make

Now move the luqix file to your global paths or share with a friend. Thank you.
