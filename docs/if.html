The if/else statement executes a block of code if a specified condition is true.

Just like other if loops, a condition is set and if its met, then the given code will execute. First we must write the if keyword followed by the condition and the demacate the loop's scope.

Now we can define a bunch if other options incase the previous ones are not fulfilled using the elif keyword. If there condition is fulfilled then it executes.

Another option is the else statement that won't base on a condition that will execute incase all the suggested conditions fail.

user = 'Rogers' // get some user input from console usr = prompt('Enter username: ') if (usr == user) print('You are logged in') elif (usr == 'ROGERS') print('You logged in will caps') else print('Wrong name, try again')

Remember that all the elif and else statement are optional.

No new scope is made and so you are simply in the parent scope just controlling the execution flow.

if true { // write some code }

This practice is good as it makes code clear and segmented.

Specialities
