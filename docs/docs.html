Know about Luqix

Luqix is object oriented and tries all its data somehow as objects.

It has highly customizable syntax thanks to the multi code demacation options inbuilt its scratch crafted lexer and parser.

Although its data is all objects and so it has attributes. Each data having an individual attribute is expensive in memory issues and so functions are provided separately to reuse them to manuplate different data so as to save memory.
