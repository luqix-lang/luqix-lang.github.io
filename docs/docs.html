What is luqix ?

Luqix (pronounced /loo-kee-ziiÀê/) is a high level interpreted dynamical and weakly typed programming language aimed to be a general purpose one and its in early development.

Luqix is written in a low level systems language dlang and utilizes dlang's interpo features with even much low level bare metal languages like embedded C so as to catalyse the process.

Luqix is not fast interms of code execution speed but just expect advancements as time passes, and you can still make use of its straight forward functionality that gets simple scripting done.

fn hello(name){ fmt = str.capitalize(name) return f'Hello {fmt}' } user = prompt('Name: ') print(hello(user))

Description: You are prompted to write your name which will be well formatted incase of all lowercase then its printed on the screen with an hello.
