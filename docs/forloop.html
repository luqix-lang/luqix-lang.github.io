Basics-for

The for loop is here too, just similar to that of the olden language.

A for loop is a control flow statement for specifying iteration, which allows code to be executed repeatedly.

Just like other for loops, here a variable must be defined and also given a avlue which is optional. Next we set a condition for the loop to base on. Then we increment or decrement the defined variable.

for(let i=0; i < 10; i++) { print(i) }

This loop adds one to the i variable and its also added to the calling scope.

++ and --

++ adds 1 to the defined variable as -- subtracts 1.

break keyword

The break keyword stops the loop from running.

for(i; i < 10; i++) { if (i == 7) break print(i) }
