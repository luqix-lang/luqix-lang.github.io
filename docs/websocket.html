Websocket module

websocket module its builtin the runtime on top of a TCP socket, this means you could create your own websocket in native luqix. Implementing one is not the easiest but easy.

Did you know?
Its very easy to gui apps with websockets and the browser, since javascript has a good support for it.

Implementing a websocket is not the easiest think at first hand due to the werd encoding it uses, but in the end its a piece of cake. So thats way it builtin the runtime but making your option could be better with even more control over it. Learn to build a websocket step by step with luqix.

Module Attributes

Websocket

websocket.Websocket()

Creates a luqix websocket object with methods to manuplate it.

server = websocket.Websocket()
attribute function
connect Binds the websocket to the given hostname and port number, connect('localhost', 7575).
close Shuts down and closes the websocket, close().
on Appends given function to the specified event, on('message', function(msg){...}). Events can be message and open, then a function with 1 argument.
start Starts the connection and only listens and accepts one client, start('1.1'), the http version is optional, it defaults to 1.1.
send Sends data to client if connection was successful, send(bytes('hello')), it should be of type bytes.
