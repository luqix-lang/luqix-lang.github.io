Console - command line app

Luqix being a real interpreter, it provides this cmd app easily without need for any hidden compilation but just in time interpreting of the given string text in the console interface.

The console app is built on simple stdin user input mechanism without need for complex libraries like ncurses atleast for now.

features usage
output highlighting for terminals supporting ascii, this makes reviews easier.
error handling and error won't terminate the program but show it in realtime.
auto multiline makes in possible to write statements.
version info quickly shows version an system info.

Usage

Just calling luqix from the terminal will open the luqix interpreter console using the commands below.

Info
Remember to have the luqix in the root path for executables or you much work in the directory with the luqix interpreter, the options are endless.

luqix

./luqix

luqix.exe

localhost~$ uname Linux localhost~$ luqix Hey, am luqix v0.9.1. At start, Type '--help' for more information >

Proceed with writing any valid code. You can import modules, calculate operations and if any return, it will be written just like interpreting a file in real time as its changed.

Hey, am luqix v0.9.1. At start, Type '--help' for more information > 7*2-4 10 > a = 'hello world' > a 'hello world' >

The functionality of this console app will improve in coming versions in the future so as to meet standards.

Starter commands

As you start the interpreter, you can follow with some arguments, that will do some other unco-ordinated things you needs to know.

argument usage
--help Returns some help for luqix executable usage.
--exec Returns path for the luqix executable.
--version Luqix version your using

Info
