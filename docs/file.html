file.Open(str filename, string mode)

Opens a file object to which you can manipulate the file data.

io = file.Open('sample.txt', 'rw')
attribute function
mode variable for the mode the app is opened in.
tell returns the position of the cursor in the file.
isopen returns true if file is still open for handling or false.
writeline writes a single line to the file at the cursor position.
name immutable variable for the file name.
size returns the realtime file size.
chunks returns true if the connection is still running or false.
read reads the given number of data from the file if not the whole file.
flush refreshes the file stream for more swift handling.
seek
write writes to the file the given data depending on the mode.
readline reads a single line of the file data.
close closes the file.
