File module

file module deals with file handling the likes of reading and writing data to files mainly. This is a convient module has it includes low level file writing with usage of bytes.

let file = require('file');

Summarised table

Function name Usage and Description
Open Opens a file in the given mode Open('file.txt', 'wb') then returns a stream object.
append Directly appends string data to the given file, append('file.txt', 'hello world').
appendbin Directly appends bytes data to the given file, append('a.mp3', bytes([100, 27])).
read Reads the whole file returning all its string data read('file.txt') without opening it.
readbin Reads the whole file returning all its bytes data read('a1.wav') or you can specify the length readbin('a1.wav', 2046) to read.
size Returns the file size size('vid.mp4') in bytes.
write For quick writing string files write('index.js', 'alert("hello world")')
writebin For quick writing bytes files writebin('pwd.zip', bytes([123, 100]))

Note
Functions dealing with strings approach things way different from those dealing with bytes. I would even prefer bytes as in the end its all bytes.

# file.Open(str filename, string mode)

Opens a file in the specified mode and returns an object stream to which you can manipulate the file data. Talking of modes, they are quite a number. These specify how the file should be handled, either in string mode or bytes mode.

modes
