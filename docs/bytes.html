Builtin module

Bytes module

bytes has functions for working with bytes data, bytes in luqix are simply an array of char[] but with this module you can also utilize ubyte data.

Module Attributes

append

index

byte_array

decode

to_byte_array

bytes.byte_array(int list)

Creates ubyte data object with attributes to manuplate their data.

bits = bytes.byte_array([45,56,78,47,23])
attribute function
append adds a ubyte number to the bytes array
pop removes and returns a char data from the byte array.
int_array converts the bytes data to intergers.
index gets a ubyte data as char data of the given index.
decode decodes the byte_array to a char array or normal luqix bytes.
