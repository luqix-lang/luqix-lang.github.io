math module

Module math

This module provides access to the mathematical functions defined by the C standard and dlang's std.math library.

# ceil (n)

returns the smallest integer that is greater than or equal to the value passed as the argument (i.e.: rounds up the nearest integer).

x = math.ceil(18.601)

# copysign(x, y)

The copysign functions return a floating-point value that combines the magnitude of x and the sign of y

n = math.copysign(34, 0-2)

# fabs(x)

returns the absolute value of the argument. It is defined in the cmath header file

n = math.fabs(0-10) // 10

# floor(x)

returns the largest integer that is smaller than or equal to the value passed as the argument (i.e.: rounds down the nearest integer)

n = math.floor(12.7) // 12

# fmod(x, y)

return the remainder(modulus) of 2 floating-point numbers mentioned in its arguments

n = math.fmod(9.6, 3.5) // 2.6
