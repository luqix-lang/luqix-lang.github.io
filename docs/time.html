Module Time

This time module is simply a wrapper of the C time.h header file. It provides almost all funtionality for the C language and returns alike data.

$ sleep(int seconds) *

Takes in seconds as an integer seconds thats seen as microseconds. This stops calling thread for the given time.

time.sleep(500) // sleeps for half a minute
