Thread module

Module thread

This module using synchronized runtime so as to support shared data during interpreter execution, this is because the language model uses this mutable data so much.

This one is very safe and so we expect good results out of the execution, well garbage collected and high level.

Module attributes.

Thread

thread_count

$ thread_count

Returns the number of all successively formed threads even after they are done executing. Includes the main thread.

threading.thread_count()

$ Thread(function fn, [args]) *

Creates a new thread object that is synchronized to prevent corruption as different threads access the same data or resource.

fn foo(fruit) print('A ', fruit, ' is sweet') th = threading.Thread(foo, ['banana'])
