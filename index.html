Readable and Explained Finabocci program

We first import list as it contains list data manuplation functions. We then make a function finabocci with parameter n that determines how many finabocci numbers should be returned.

We create var a, b, c and list fib to which be add a, b and c as the starters, and then a for loop residing on the n parameter. As the code runs, a becomes b and b becomes c as c becomes a + b which is then appened to the fib list.

Then the fib list is returned thats then printed onto the console.
