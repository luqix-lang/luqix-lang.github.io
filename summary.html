#Documentation

Just some brief documentation, Luqix is an easy to use programming language, has very clean syntax and supports multi code multi code demacation options just to write in your way.

#Syntax

We follow the C-style syntax. Demacated with indentation or curl-braces {}. Mind the indentation to prevent uncaught errors.

// decmacting with culr-braces for(let i=0; i<10; i++) { print(i) } scores = 10 // indenting with tabs while (scores) print(scores) scores = scores-1

#Functions

For any reasonable program, there are functions, they help in code flow, reusing code, and determining when and how a task if to execute. A function has a name, parameters and code demacated to it, it's after then called anywhere accessable in the code scope.

A function will create and keep its own scope but its able to return data using the return keyword.

fn random_game() { print('what is the bravest animal') guess = 'lion' ans = prompt('1 chance: ') if (ans == guess) return 'You win' return 'You lost' } print(random_game())

#Classes

Classes are used to create personal datatypes/objects, you will give them attributes and you are not limited to your modification, inheritance is supported.

A function will create and keep its own scope but its able to return data using the return keyword.

class Student{ fn __init__(name, level) { self.name = name self.level = level } fn update(n, l) { self.name = n self.level = l } } school = [Student('Ronnie', 5), Student('Brian', 1), Student('Cate', 4), Student('Babs', 3)] // accessing atributes for(let i; i < len(school); i++) print(f'My name is {school[i].name} and am in form {school[i].level}.')

#Locals

This is one super usefull module in Luqix has it has core functions to do things like find data length, output on the console, exit the current thread and may more.

Luckily, it is automatically imported in every new scope due to its needy usage of its functions.

Outputs given data on the console.

print('hello world') // hello world

Safely imports core modules to the calling scope.

rd = importc('random') // 4

Used to find the length of a datatype.

len([1,2,3,4]) // 4

Lists the attributes of any datatype

attr(enum{a=1, b=2}) // [a, b]

Get user input from the console.

prompt('Username: ')

Returns the type of data as a string.

type('hello world') // string

Creates a genuine exception understood by the interpreter error handler.

Exception(enum{msg='undefined', tyep='referenceError', line=0})

Gets an attribute from given datatype without having to use the . character.

getattr(enum{a=1}, 'a') // 1

Removes attribute specified attribute from the given data.

delattr(enum{a=1, b=2}, 'b')

Sets a new or replaces attribute of a given datatype.

a = enum{a=1} setattr(a, 'b', 2)

Used in inheriting a class.

super(_class, 'attr')
